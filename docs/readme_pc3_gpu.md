# PC3.GL GPU
Viewer helps observing 3-arrays in real-time using GPU and fluid rendering.

![pc3 viewer](docs/pc3.gl.png "pc3 viewer")

## Keys
| key   | description | 
| ------------ | ------------ |
|A    | LEFT      | 
|D    | RIGHT     | 

### Example
```
import pc3.gpu as PC3

class App(PC3):
    def __init__(super):
        pass
App.run()

```
